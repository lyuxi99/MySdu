# 功能
* 课程表：随时随地看课表，点击课程还可添加备注
* 待办事项：方便记录作业，不再忘记
* 校区地图：精确定位，高清地图，校园内不再迷路
* 成绩查询：一键查询成绩，心痛只在瞬间
* 身份证二维码：核酸检测时方便展示，不再需要打开小程序
* 出入校园信息填报：一键打开出校申请
* 图书馆预约：一键打开座位预约
* 空教室查询：方便查询空教室，自习必备
* 考试安排：<待开发>
